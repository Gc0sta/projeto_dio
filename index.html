História da Web

Como surgiu?

O surgimento da internet teve seu início em 1949, os Estados Unidos e a Rússia protagonizavam um embate de termos ideológicos, econômicos,militares, políticos e tecnológicos. Tal embate ficou conhecido como Guerra Fria

Em 1960 Joseph Carl teorizou sobre uma rede onde todos poderiam se comunicar, neste momento uma semente foi plantada. Foi desenvolvido então um sistema de comunicação em pacotes que ao interligar múltiplos pontos de interseccção, permitia o envio de pacotes entre si.

Tim Berners-Lee em 1989, uniu o hipertexto com TCP/IP criando assim a World Wide Web.

Oque são Clients?

Os clients podem ser definidos como os dispositivos que realizam a solicitação ou iniciam a comunicação com o servidor. Após a conexão ser estabelecida o servidor realiza o envio dos arquivos estáticos para o client. O navegador então armazena esses dados em um cache no computador para que não seja necessário uma nova busca pelos arquivos no servidor.

Os navegadores são também chamados de "browsers", são programas criado por empresas para abrir ou executar arquivos.

Oque são servidores?

São computadores de grande procesamento que armazenam arquivos e distribuem de acordo com a requisição feita.Existem diversos tipos de servidores, entre eles temos o Servidor de arquivos que podem funcionar sem internet e permitem que multiplos usuários possam ter acesso a determinado arquivo. Os servidores web são voltados para internet e são responsáveis por controlar como os usuários acessam os arquivos hospedados.

Linguagens de Programação

A Linguagem de programação pode ser resumida como uma linguagem escrita e formal que define um conjuto de instruções para gerar programas. Existem linguagens que são "Server-side" e "Client-side", como o proprio nome sugere as linguagens Server-side são executadas no servidor. As linguagens de Client-side são as linguagens que são executadas no navegador do cliente. Um exemplo de linguagem é javascript que pode rodar dos dois lados, tudo isso só é capaz graças a uma engine chamada v8 e Node.js no caso de servidores.

Introdução ao Html na Prática

Estrutura Básica do Html

O Html é formado por 3 partes, a tag html, a cabeça e o corpo.A tag head contem os metadados, título e codificação de caracteres na página, essa informações serão pre-carregadas antes que a página seja exibida ao usuário, a tag body contem tudo que será exibido para o usuário.

As tags são elementos que definem o conteúdo da página, elas indicam para os navegadores como o conteúdo deve ser exibido.As tags podem conter atributos, elas são propriedades que um tag pode ter. Um exemplo de atributo é a tag "id=" ela serve como identificador da tag sendo possível realizar a alteração deste elemento de forma precisa.

Textos

Na construção dos textos do site existem diversas tags que podem ser utilizadas, uma dela é a tag <h1> que significa heading 1, ele atua como um título para a página enquanto o <h2> atua como um subtítulo. A tag "h" vai de <h1> até <h6>, outra tag que podemos utilizar é a <Blockquote> que serve para fazermos citações.Para a criação de parágrafos utilizamos a tag <p>, dentro dela todo texto que for escrito será parte do parágrafo, diferente de como estamos acostumados,dentro da tag <p> os espaços em branco não são considerados, dessa forma mesmo que possua uma quebra de linha ela não irá ser exibida ao usuário, a menos que utilize a tag <br> que significa "break", ou pode utilizar propriedades do css.

Listas

Existem dois tipos de listas em html são elas:
<ol> (Lista ordenada)
<ul> (Lista não ordenada)
A diferença entre elas é que na lista ordenada a ordem importa, ela pode ser utilizada por exemplo em listas de espera enquanto as listas não ordenadas atuam como listas comuns, elas podem ser usadas em menus por exemplo.

Links

Metaforicamente falando links são como ligações entre sites, eles podem te direcionar de um local para outro. Em html utilizamos a tag <a> que significa "anchor" ou "âncora", com ela podemos realizar a criação de links. Da mesma forma que outras tags ela também possui atributos, porém eles não se categorizam como atributos globais, um deles é o target="_blank".
